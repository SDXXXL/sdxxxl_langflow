<h2 id="why-use-langgraph" class="tsd-anchor-link">Why use LangGraph?<a href="#why-use-langgraph" aria-label="Permalink" class="tsd-anchor-icon"><svg viewBox="0 0 24 24" aria-hidden="true"><use href="../assets/icons.svg#icon-anchor"></use></svg></a></h2><p>LangGraph is built for developers who want to build powerful, adaptable AI agents. Developers choose LangGraph for:</p>
<ul>
<li><strong>Reliability and controllability.</strong> Steer agent actions with moderation checks and human-in-the-loop approvals. LangGraph persists context for long-running workflows, keeping your agents on course.</li>
<li><strong>Low-level and extensible.</strong> Build custom agents with fully descriptive, low-level primitives ‚Äì free from rigid abstractions that limit customization. Design scalable multi-agent systems, with each agent serving a specific role tailored to your use case.</li>
<li><strong>First-class streaming support.</strong> With token-by-token streaming and streaming of intermediate steps, LangGraph gives users clear visibility into agent reasoning and actions as they unfold in real time.</li>
</ul>
<p>LangGraph is trusted in production and powering agents for companies like:</p>
<ul>
<li><a href="https://blog.langchain.dev/customers-klarna/" target="_blank" class="external">Klarna</a>: Customer support bot for 85 million active users</li>
<li><a href="https://www.elastic.co/blog/elastic-security-generative-ai-features" target="_blank" class="external">Elastic</a>: Security AI assistant for threat detection</li>
<li><a href="https://dpe.org/sessions/ty-smith-adam-huda/this-year-in-ubers-ai-driven-developer-productivity-revolution/" target="_blank" class="external">Uber</a>: Automated unit test generation</li>
<li><a href="https://www.langchain.com/breakoutagents/replit" target="_blank" class="external">Replit</a>: Code generation</li>
<li>And many more (<a href="https://www.langchain.com/built-with-langgraph" target="_blank" class="external">see list here</a>)</li>
</ul>
<h2 id="langgraphs-ecosystem" class="tsd-anchor-link">LangGraph‚Äôs ecosystem<a href="#langgraphs-ecosystem" aria-label="Permalink" class="tsd-anchor-icon"><svg viewBox="0 0 24 24" aria-hidden="true"><use href="../assets/icons.svg#icon-anchor"></use></svg></a></h2><p>While LangGraph can be used standalone, it also integrates seamlessly with any LangChain product, giving developers a full suite of tools for building agents. To improve your LLM application development, pair LangGraph with:</p>
<ul>
<li><a href="http://www.langchain.com/langsmith" target="_blank" class="external">LangSmith</a> ‚Äî Helpful for agent evals and observability. Debug poor-performing LLM app runs, evaluate agent trajectories, gain visibility in production, and improve performance over time.</li>
<li><a href="https://langchain-ai.github.io/langgraphjs/concepts/#langgraph-platform" target="_blank" class="external">LangGraph Platform</a> ‚Äî Deploy and scale agents effortlessly with a purpose-built deployment platform for long running, stateful workflows. Discover, reuse, configure, and share agents across teams ‚Äî and iterate quickly with visual prototyping in <a href="https://langchain-ai.github.io/langgraphjs/concepts/langgraph_studio/" target="_blank" class="external">LangGraph Studio</a>.</li>
</ul>
<h2 id="pairing-with-langgraph-platform" class="tsd-anchor-link">Pairing with LangGraph Platform<a href="#pairing-with-langgraph-platform" aria-label="Permalink" class="tsd-anchor-icon"><svg viewBox="0 0 24 24" aria-hidden="true"><use href="../assets/icons.svg#icon-anchor"></use></svg></a></h2><p>While LangGraph is our open-source agent orchestration framework, enterprises that need scalable agent deployment can benefit from <a href="https://langchain-ai.github.io/langgraphjs/concepts/langgraph_platform/" target="_blank" class="external">LangGraph Platform</a>.</p>
<p>LangGraph Platform can help engineering teams:</p>
<ul>
<li><strong>Accelerate agent development</strong>: Quickly create agent UXs with configurable templates and <a href="https://langchain-ai.github.io/langgraphjs/concepts/langgraph_studio/" target="_blank" class="external">LangGraph Studio</a> for visualizing and debugging agent interactions.</li>
<li><strong>Deploy seamlessly</strong>: We handle the complexity of deploying your agent. LangGraph Platform includes robust APIs for memory, threads, and cron jobs plus auto-scaling task queues &amp; servers.</li>
<li><strong>Centralize agent management &amp; reusability</strong>: Discover, reuse, and manage agents across the organization. Business users can also modify agents without coding.</li>
</ul>
